LV Human-Computer Interaction (BSc)

  • 4th semester, ≈250 students, in groups of 4, in English.
  • 4.5 ECTS
  • https://courses.isds.tugraz.at/hci/

Thinking Aloud Testing

  • ≈5 test users think out loud, while working with an interface.
  • Then, ask them what they thought.
  • For example:
    https://courses.isds.tugraz.at/hci/reports/ss2017/g4-08-graz-de/ta/ta-de.html

LV Information Architecture and Web Usability (MSc)

  • ≈16-20 students, in groups of 4, in English.
  • 5 ECTS
  • https://courses.isds.tugraz.at/iaweb/

Tree Testing

  • Testing candidate hierarchies, e.g. product categories or menus.
  • Ask a group of users where they would expect to find things.

Tree Testing with Diamond

  • Web-based tree testing and card sorting.
  • Mathias Blum, ongoing [and several previous students].
  • Code: https://github.com/tugraz-isds/diamond
  • Demo: local

Responsive Web Design

  • Web site responds dynamically to characteristics of device.
  • The Boston Globe (bostonglobe.com) was first responsive web site (2011).
  • Nowadays, web design is responsive web design!

LV Information Visualisation (MSc)

  • ≈16-20 students, in groups of 4, in English.
  • 5 ECTS
  • https://courses.isds.tugraz.at/ivis/

Multidimensional Analysis

  • 73 cities, 56 indices (dimensions), from 2012.
  • Macrofocus InfoScope.

Master's Thesis

  • Typically towards then end of the Master's degree.
  • 30 ECTS.
  • For example:
    • Rslidy: Responsive Slide Decks.
    • VoroTree: Voronoi Treemaps in JavaScript.
    • Hypertree: Hyperbolic Tree Browser in JavaScript.

Rslidy: Responsive Slide Decks

  • Web-based, responsive slides.
  • Patrick Hipp, Master's Thesis, 2021 [and several previous students]. https://ftp.isds.tugraz.at/pub/theses/phipp-2021-msc.pdf
  • Code: https://github.com/tugraz-isds/rslidy
  • Demo: this slide deck...

VoroTree: Voronoi Treemaps in JavaScript

  • Navigation through a hierarchy using recursive Voronoi subdivision.
  • Chris Oser, Master's Thesis, 2022 [and several previous students]. https://ftp.isds.tugraz.at/pub/theses/coser-2022-msc.pdf
  • Code: https://github.com/somestudentcoder/vorotree
  • Demo: https://somestudentcoder.github.io/vorotree/

Hypertree: Hyperbolic Tree Browser in JavaScript

  • Navigation through a hierarchy using hyperbolic geaometry.
  • Michael Glatzhofer, Master's project, 2018.
  • Code: https://github.com/glouwa/d3-hypertree
  • Demo: Product Hierarchy https://keithandrews.com/demos/ivis/hypertree/hypertree-products/
  • Demo: Hyperbolic Tree of Life https://hyperbolic-tree-of-life.github.io/
