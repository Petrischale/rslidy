About rslidy


https://github.com/tugraz-isds/rslidy
  • A way to design responsive and lightweigth presentations in html5, css3 and TypeScript.
  • Runs directly in the web browser.
  • Open source.
  • Offers a variety of options for presentations in html:
    1. Possibility of slide navigation.
    2. Possibility of slide overview.
    3. Enables a responsive design.
    4. Enables responsive interactions (incl. shake and tilt).
    5. Supports CSS styles for a few options (Animations, Fonts).
    6. Allows embedding of live code.
    7. Image viewer (incl. zoom and pan).

Project ideas


  • Scoping to hide slidy's internal css.
  • Include container queries.
  • Include live code.

Scoping to hide slidy's internal css

  • This would allow to adapt the status bar at the bottom.
  • Design options could be similar to PowerPoint.
  • Offer the user a variety of design options to choose from.
  • Possible by scoping and hiding the original css.

Inclusion of container queries

  • Would allow a design similar to PowerPoint.
  • User could create containers on the slides.
  • Possibility to offer basic design patterns for slides.
  • Open question: how much effort to include this in slidy.

Flawless inclusion of live code

  • Would come with benefits for the user.
  • We have not found good ways to do so by now.
  • Open question: What feature would you like to have?
  • Open question: Is there a nice way?
